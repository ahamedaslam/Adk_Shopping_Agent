# Adk_Shopping_Agent

### Windows

1. create python environment - python -m venv venv
2. activate the env - venv\Scripts\activate
3. if we create any dependencies it will create inside the env instead of global system


### Install dependencies

1. pip install --> upgrade google-adk
2. pip show google-adk ---> To check adk installed
3. adk web --> to run the app
